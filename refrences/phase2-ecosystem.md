# Ecosystem Design - Abjad Programming Language

**المرحلة:** ٢ - النظام البيئي (Ecosystem)
**التاريخ:** Q3 2025

---

## ١. نظرة عامة (Overview)

النظام البيئي في أبجد يوفر أدوات لبناء وإدارة وتوزيع المشاريع. يشمل مدير الحزم، نظام البناء، ومولد التوثيق.

The ecosystem in Abjad provides tools for building, managing, and distributing projects. It includes a package manager, build system, and documentation generator.

---

## ٢. مدير الحزم (Package Manager)

### ٢.١ هيكل الحزمة (Package Structure)

```
اسم_الحزمة/
├── أبجد.تومل          # ملف تعريف الحزمة
├── المصدر/            # الكود المصدري
│   └── رئيسي.أبجد     # الملف الرئيسي
├── الاختبارات/        # الاختبارات
├── التوثيق/           # التوثيق
└── أمثلة/            # الأمثلة
```

### ٢.٢ ملف أبجد.تومل (abjad.toml)

```toml
[حزمة]
اسم = "مثال"
إصدار = "٠.١.٠"
مؤلفون = ["فريق أبجد"]
وصف = "حزمة مثال لأبجد"
رخصة = "MIT"

[تبعيات]
أبjad = "٠.١.٠"
مجموعات = "٠.١.٠"

[تبعيات.تطوير]
اختبارات = "٠.١.٠"

[بناء]
هدف = "مثال"
نوع = "تنفيذي"

[أهداف]
[أهداف.إصدار]
مسار = "target/release"
```

### ٢.٣ أوامر مدير الحزم (Package Manager Commands)

#### إضافة حزمة (Add Package)
```bash
أبجد إضافة اسم_الحزمة
```

#### إزالة حزمة (Remove Package)
```bash
أبجد إزالة اسم_الحزمة
```

#### تحديث الحزم (Update Packages)
```bash
أبجد تحديث
```

#### تثبيت الحزم (Install Packages)
```bash
أبjad تثبيت
```

#### نشر حزمة (Publish Package)
```bash
أبjad نشر
```

#### البحث عن حزمة (Search Package)
```bash
أبjad بحث اسم_الحزمة
```

## ٣. نظام البناء (Build System)

### ٣.١ ملفات البناء (Build Files)

```toml
[بناء]
تجميع = "true"
تحسين = "٣"
فحص = "true"

[بناء.أهداف]
[بناء.أهداف.مثال]
مسار = "المصدر/رئيسي.أبجد"
نوع = "تنفيذي"

[بناء.أهداف.مكتبة]
مسار = "المصدر/مكتبة.أبجد"
نوع = "مكتبة"
```

### ٣.٢ أوامر البناء (Build Commands)

#### بناء المشروع (Build Project)
```bash
أبjad بناء
```

#### بناء مع إصدار (Build with Release)
```bash
أبjad بناء --إصدار
```

#### بناء مع تحسين (Build with Optimization)
```bash
أبjad بناء --تحسين ٣
```

#### تنظيف البناء (Clean Build)
```bash
أبjad بناء --تنظيف
```

## ٤. مولد التوثيق (Documentation Generator)

### ٤.١ تعليقات التوثيق (Documentation Comments)

```abjad
/// هذه دالة مثال
/// 
/// # المعاملات
/// 
/// * `أ` - الرقم الأول
/// * `ب` - الرقم الثاني
/// 
/// # القيمة المُرجعة
/// 
/// مجموع الرقمين
دالة جمع(أ: صحيح٣٢، ب: صحيح٣٢) -> صحيح٣٢ {
    أعد أ + ب
}
```

### ٤.٢ أوامر التوثيق (Documentation Commands)

#### توليد التوثيق (Generate Documentation)
```bash
أبjad توثيق
```

#### فتح التوثيق (Open Documentation)
```bash
أبjad توثيق --فتح
```

#### نشر التوثيق (Publish Documentation)
```bash
أبjad توثيق --نشر
```

## ٥. نظام الاختبارات (Testing System)

### ٥.١ كتابة الاختبارات (Writing Tests)

```abjad
اختبار اختبار_الجمع() {
    دع نتيجة = جمع(١، ٢)
    تأكيد(نتيجة == ٣)
}
```

### ٥.٢ أوامر الاختبارات (Test Commands)

#### تشغيل الاختبارات (Run Tests)
```bash
أبjad اختبار
```

#### تشغيل اختبار محدد (Run Specific Test)
```bash
أبjad اختبار اختبار_الجمع
```

#### تشغيل الاختبارات مع التغطية (Run with Coverage)
```bash
أبjad اختبار --تغطية
```

## ٦. إدارة الإصدارات (Version Management)

### ٦.١ نظام الترقيم الدلالي (Semantic Versioning)

```
الرئيسي.الثانوي.التصحيح
```

- **الرئيسي (Major):** تغييرات غير متوافقة
- **الثانوي (Minor):** إضافة ميزات متوافقة
- **التصحيح (Patch):** إصلاحات أخطاء

### ٦.٢ أوامر الإصدارات (Version Commands)

#### إنشاء إصدار (Create Release)
```bash
أبjad إصدار --نوع رئيسي
```

#### عرض الإصدارات (Show Versions)
```bash
أبjad إصدار --عرض
```
