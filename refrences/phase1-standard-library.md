# Standard Library Design - Abjad Programming Language

**المرحلة:** ١ - المكتبة القياسية (Standard Library)
**التاريخ:** Q2 2025

---

## ١. نظرة عامة (Overview)

المكتبة القياسية في أبجد توفر وظائف أساسية ومجموعات بيانات وعمليات إدخال/إخراج. تتميز بالدعم الكامل للغة العربية والاتجاه من اليمين إلى اليسار.

The standard library in Abjad provides core functions, data collections, and I/O operations. It features full Arabic support and right-to-left direction.

---

## ٢. هيكل المكتبة (Library Structure)

```
أبجد/
├── أساسيات/          # Core
│   ├── أنواع          # Types
│   ├── أرقام          # Numbers
│   ├── منطق           # Logic
│   └── سلاسل          # Strings
├── مجموعات/          # Collections
│   ├── مصفوفات        # Arrays
│   ├── قوائم          # Lists
│   ├── خرائط          # Maps
│   └── مجموعات        # Sets
├── إدخال_إخراج/       # I/O
│   ├── ملفات          # Files
│   ├── وحدات          # Streams
│   └── طرفيات         # Terminals
├── تزامن/             # Concurrency
│   ├── خيوط          # Threads
│   ├── قنوات          # Channels
│   └── أقفال          # Locks
└── أخطاء/             # Errors
    ├── أنواع          # Types
    ├── معالجة         # Handling
    └── استرداد        # Recovery
```

## ٣. الأساسيات (Core)

### ٣.١ الأنواع (Types)

#### النوع `أي` (Any)
```abjad
// يمكن أن يمثل أي قيمة
دالة معالجة(قيمة: أي) {
    // معالجة القيمة
}
```

#### النوع `خيار` (Option)
```abjad
// يمثل قيمة موجودة أو غير موجودة
دالة قد_تكون(قيمة: صحيح٣٢) -> خيار<صحيح٣٢> {
    إذا (قيمة > ٠) {
        بعض(قيمة)
    } آخر {
        لا_شيء
    }
}
```

#### النوع `نتيجة` (Result)
```abjad
// يمثل نجاح أو فشل
دالة تقسيم(أ: صحيح٣٢، ب: صحيح٣٢) -> نتيجة<صحيح٣٢، خطأ> {
    إذا (ب == ٠) {
        خطأ("لا يمكن القسمة على صفر")
    } آخر {
        نجاح(أ / ب)
    }
}
```

### ٣.٢ الأرقام (Numbers)

#### العمليات الحسابية (Arithmetic Operations)
```abjad
استورد "أبجد/أساسيات/أرقام"

دالة مثال() {
    متغير أ = ١٠
    متغير ب = ٥
    
    متغير جمع = أ + ب
    متغير طرح = أ - ب
    متغير ضرب = أ * ب
    متغير قسمة = أ / ب
    متغير باقي = أ % ب
}
```

#### العمليات البتية (Bitwise Operations)
```abjad
استورد "أبجد/أساسيات/أرقام"

دالة مثال() {
    متغير أ = ١٠
    متغير ب = ٥
    
    متغير و_بت = أ & ب
    متغير أو_بت = أ | ب
    متغير xor_بت = أ ^ ب
    متغير إزاحة_يسار = أ << ١
    متغير إزاحة_يمين = أ >> ١
}
```

### ٣.٣ المنطق (Logic)

#### القيم المنطقية (Boolean Values)
```abjad
استورد "أبجد/أساسيات/منطق"

دالة مثال() {
    متغير صحيح = صحيح
    متغير خطأ = خطأ
    
    متغير و_منطقي = صحيح و خطأ
    متغير أو_منطقي = صحيح أو خطأ
    متغير ليس = ليس صحيح
}
```

### ٣.٤ السلاسل (Strings)

#### إنشاء السلاسل (String Creation)
```abjad
استورد "أبجد/أساسيات/سلاسل"

دالة مثال() {
    متغير نص = "مرحباً بالعالم"
    متغير نص_فارغ = ""
    
    متغير طول = نص.طول()
    متغير فارغ = نص.فارغ()
}
```

#### عمليات السلاسل (String Operations)
```abjad
استورد "أبجد/أساسيات/سلاسل"

دالة مثال() {
    متغير نص = "مرحباً"
    
    متغير مضاف = نص + " بالعالم"
    متغير يحتوي = نص.يحتوي("مر")
    متغير يبدأ_بـ = نص.يبدأ_بـ("م")
    متغير ينتهي_بـ = نص.ينتهي_بـ("ا")
}
```

#### تحويل السلاسل (String Conversion)
```abjad
استورد "أبجد/أساسيات/سلاسل"

دالة مثال() {
    متغير نص = "١٢٣"
    
    متغير رقم = نص.إلى_رقم()
    متغير نص_من_رقم = ٤٥٦.إلى_نص()
}
```

## ٥. إدخال/إخراج (I/O)

### ٥.١ الملفات (Files)

#### قراءة الملفات (File Reading)
```abjad
استورد "أبجد/إدخال_إخراج/ملفات"

دالة مثال() {
    متغير ملف = ملف_جديد("example.abjad")
    متغير محتوى = ملف.اقرأ_الكل()
    
    طباعة(محتوى)
}
```

#### كتابة الملفات (File Writing)
```abjad
استورد "أبجد/إدخال_إخراج/ملفات"

دالة مثال() {
    متغير ملف = ملف_جديد("output.txt")
    ملف.اكتب("مرحباً بالعالم")
}
```

```abjad
استورد "أبجد/إدخال_إخراج/وحدات"

دالة مثال() {
    متغير وحد = وحد_إدخال_قياسي()
    متغير سطر = وحد.اقرأ_سطر()

}
```

#### وحدات الإخراج (Output Streams)
```abjad
استورد "أبجد/إدخال_إخراج/وحدات"

دالة مثال() {
    متغير وحد = وحد_إخراج_قياسي()
    
    وحد.اكتب("مرحباً بالعالم")

}
```

### ٥.٣ الطرفيات (Terminals)

## ٦. التزامن (Concurrency)

### ٦.١ الخيوط (Threads)

#### إنشاء الخيوط (Thread Creation)
```abjad
استورد "أبجد/تزامن/خيوط"

دالة مهمة() {
    طباعة("الخيط يعمل")
}

دالة مثال() {
    متغير خيط = خيط_جديد(مهمة)
    خيط.ابدأ()
    خيط.انتظر()
}
```

### ٦.٢ القنوات (Channels)

#### إنشاء القنوات (Channel Creation)
```abjad
استورد "أبجد/تزامن/قنوات"

دالة منتج(قناة: قناة<صحيح٣٢>) {
    لكل رقم في ٠..١٠ {
        قناة.أرسل(رقم)
    }
}

دالة مستهلك(قناة: قناة<صحيح٣٢>) {
    حلقة {
        متغير قيمة = قناة.استقبل()
        طباعة(قيمة)
    }
}
```

### ٦.٣ الأقفال (Locks)

#### استخدام الأقفال (Lock Usage)
```abjad
استورد "أبجد/تزامن/أقفال"

هيكل عداد {
    قيمة: صحيح٣٢،
    قفل: قفل_متزامن،
}

دالة مثال() {
    متغير عداد = عداد_جديد(٠)
    
    عداد.قفل.قفل()
    عداد.قيمة += ١
    عداد.قفل.إلغاء_القفل()
}
```

## ٧. الأخطاء (Errors)

### ٧.١ أنواع الأخطاء (Error Types)

#### إنشاء أنواع الأخطاء (Custom Error Types)
```abjad
استورد "أبjad/أخطاء/أنواع"

خطأ خطأ_القسمة {
    رسالة: نص،
}

دالة تقسيم(أ: صحيح٣٢، ب: صحيح٣٢) -> نتيجة<صحيح٣٢، خطأ_القسمة> {
    إذا (ب == ٠) {
        خطأ("لا يمكن القسمة على صفر")
    } آخر {
        نجاح(أ / ب)
    }
}
```

### ٧.٢ معالجة الأخطاء (Error Handling)

#### معالجة الأخطاء (Error Handling)
```abjad
استورد "أبjad/أخطاء/معالجة"

دالة مثال() {
    متغير نتيجة = تقسيم(١٠، ٠)
    
    تطابق نتيجة {
        نجاح(قيمة) => طباعة("النتيجة: " + قيمة),
        خطأ(رسالة) => طباعة("خطأ: " + رسالة),
    }
}
```

### ٧.٣ استرداد الأخطاء (Error Recovery)

#### استرداد من الأخطاء (Error Recovery)
```abjad
استورد "أبjad/أخطاء/استرداد"

دالة مثال() {
    محاولة {
        تقسيم(١٠، ٠)
    } أو {
        طباعة("حدث خطأ، استرداد")
        نجاح(٠)
    }
}
```
