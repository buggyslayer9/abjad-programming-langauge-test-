01
الرؤية والأهداف
Vision & Objectives

أبجد هي أول لغة برمجة عربية كاملة — مكتوبةً وهيكلاً وأدواتٍ — تُجمَّع مباشرةً إلى لغة التجميع (Assembly) وتنتج كوداً أصيلاً بأداء مقارب لـ C وRust وGo. الهدف ليس مجرد ترجمة الإنجليزية إلى العربية، بل بناء نموذج جديد للتعبير البرمجي يخدم أكثر من 400 مليون متحدث عربي.

تُصمَّم اللغة لتكون سهلة التعلم كـ Python، سريعة كـ Rust، مستقرة كـ Go، مع دعم كامل لاستدعاء مكتبات C وGo وRust مباشرةً دون تكاليف إضافية.

المبدأ الجوهري: الكود العربي ليس مجرد واجهة — هو البنية كلها. المتغيرات، الدوال، الأنواع، التعليقات، الأخطاء، التوثيق — كل شيء يُكتب بالعربية الفصحى أو العامية المعيارية.

02
الفلسفة التصميمية
Design Philosophy

تقوم أبجد على سبعة مبادئ تصميمية تحكم كل قرار في اللغة والمُجمِّع والنظام البيئي:

١ · الوضوح قبل الإيجاز

الكود الواضح أهم من الكود القصير. لا تُقتل الكلمات المفتاحية، ولا تُختصر الأسماء قسراً. العربية لغة غنية بالمفردات — استخدامها كاملةً يزيد القراءة لا يُعيقها.

٢ · صفر تكلفة التجريد (Zero-Cost Abstractions)

كل تجريد في اللغة — الأنواع العامة، المشابهات (Traits)، البرمجة الوظيفية — يُحوَّل إلى كود مكافئ لما سيكتبه المبرمج يدوياً بلغة التجميع. لا عبء وقت التشغيل أبداً.

٣ · العربية أولاً، الترجمة ثانياً

ليست أبجد لغة مُترجَمة من الإنجليزية. كل كلمة مفتاحية اختُيرت من القاموس العربي لتحمل معناها بدقة. دالة أوضح من func لأي مبتدئ عربي.

٤ · الأخطاء كمواطنين من الدرجة الأولى

لا استثناءات مخفية. كل دالة قد تُخطئ تُعلن عن ذلك في نوعها. المُجمِّع يرفض تجاهل الأخطاء بصمت.

٥ · مُجمِّع واحد، منصات متعددة

تُجمَّع أبجد إلى x86-64، ARM64، RISC-V، وWebAssembly من نفس الكود المصدري دون تعديل.

٦ · التوافق كميزة أصيلة

استدعاء مكتبات C ليس إضافةً — هو جزء من اللغة الأساسية. يمكن استخدام أي مكتبة C كمكتبة أبجد مباشرةً.

٧ · الحتمي الافتراضي

البرامج تعمل بنفس الطريقة في كل مرة على كل منصة. لا سلوك غير محدد. لا مفاجآت.

أ · الأنواع العامة (Generics)

// دالة عامة تعمل على أي نوع يدعم المقارنة دالة أكبر<ن: قابل_للمقارنة>(أ: ن، ب: ن) -> ن { إذا أ > ب { أعد أ } وإلا { أعد ب } } // قائمة مكدس عامة هيكل مكدس<ن> { بيانات: قائمة[ن] } نفِّذ <ن> مكدس<ن> { دالة ادفع(&متغير ذات، قيمة: ن) { ذات.بيانات.أضف(قيمة) } دالة اسحب(&متغير ذات) -> ربما[ن] { ذات.بيانات.سحب_آخر() } }

ب · المشابهات (Traits / Interfaces)

واجهة قابل_للطباعة { دالة اطبع(&ذات) -> نص } واجهة قابل_للتسلسل { دالة إلى_يسونل(&ذات) -> نتيجة[نص، خطأ] دالة من_يسونل(بيانات: نص) -> نتيجة[ذات، خطأ] } // تنفيذ متعدد الواجهات نفِّذ قابل_للطباعة لـ نقطة { دالة اطبع(&ذات) -> نص { أعد شكّل("({}, {})", ذات.س، ذات.ص) } }

ج · الإغلاقات والبرمجة الوظيفية

متغير أرقام: قائمة[صحيح٣٢] = [١، ٢، ٣، ٤، ٥، ٦] // تصفية + تحويل + تجميع بأسلوب وظيفي سلس متغير نتيجة = أرقام .صفِّ(|ن| ن % ٢ == ٠) // الأزواج فقط: [2،4،6] .حوِّل(|ن| ن * ن) // تربيع: [4،16،36] .اجمع(٠) // مجموع: 56 // إغلاق يلتقط المتغيرات من السياق متغير عتبة = ١٠ متغير فوق_العتبة = أرقام.صفِّ(|ن| ن > عتبة)

د · نموذج الممثلين (Actor Model)

هيكل خادم_ويب { عدد_الطلبات: طبيعي٦٤ } نفِّذ ممثل لـ خادم_ويب { نوع الرسالة = طلب_ويب دالة غير_متزامن عالج_رسالة(&متغير ذات، رسالة: طلب_ويب) { ذات.عدد_الطلبات += ١ انتظر ذات.أرسل_رد(رسالة.معرف، "200 OK") } } دالة غير_متزامن رئيسية() { متغير خادم = خادم_ويب::ابدأ(خادم_ويب { عدد_الطلبات: ٠ }) انتظر خادم.أرسل(طلب_ويب { معرف: ١، مسار: "/" }) }

إدارة الذاكرة
Memory Management

تعتمد أبجد نموذجاً هجيناً لإدارة الذاكرة يجمع بين مزايا عدة أنظمة دون عيوبها:

👑
نظام الملكية
كل قيمة لها مالك واحد. عند خروج المالك من النطاق تُحرَّر الذاكرة تلقائياً — بدون GC.
📎
الاستعارة (Borrowing)
مراجع غير قابلة للتعديل متعددة، أو مرجع قابل للتعديل واحد — يُفحص وقت الترجمة.
🔢
تعداد المراجع
نوع مشترك[ن] يوفر ملكية مشتركة بتعداد مراجع ذري (Arc-like).
لا GC، لا malloc/free يدوي: المُجمِّع يُدرج تعليمات تحرير الذاكرة في المواقع الصحيحة تلقائياً بناء على تحليل الملكية. وقت توقف صفر (Zero pause times).
دالة مثال_الملكية() { متغير أ = قائمة::من([١، ٢، ٣]) // 'أ' يملك القائمة متغير ب = أ // الملكية تنتقل إلى 'ب' // طباعة(أ) ← خطأ وقت الترجمة: 'أ' فقد ملكيته متغير مرجع = &ب // استعارة غير قابلة للتعديل طباعة("الطول: {}", مرجع.طول) // يُحرَّر 'ب' هنا تلقائياً — بدون free() يدوي } دالة مثال_مشترك() { متغير مشترك_١ = مشترك::جديد(قيمة_ضخمة()) متغير مشترك_٢ = مشترك_١.استنسخ() // زيادة تعداد المراجع // يُحرَّر عند وصول التعداد إلى صفر }
